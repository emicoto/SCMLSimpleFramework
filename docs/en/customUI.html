CustomPopup Usage and Examples

Overview

CustomPopup is a utility class for Mod creators to create custom popups in the game. With this tool, you can easily create and manage popups to display the desired information or UI elements. CustomPopup provides functions to create popups, set popup size, update content, show and close popups, and supports custom CSS styles to adjust the appearance of the popups.

This document explains the features and specific use cases of CustomPopup in a beginner-friendly manner, including how to use the default CSS and how to customize the style of the popups.

Main Functionality

1. Create Popup

drawPopup(title, content, style, noBtn)

Description: Draws a popup with a title and content.

  • Parameters:
    • title (string): The title of the popup.
    • content (string): The content of the popup, can be an HTML string.
    • style (string): Custom style class name, default is 'custom-popup'.
    • noBtn (boolean): Whether to hide the close button, default is false (show close button).
  • Example:
CustomPopup.draw('Welcome', 'Welcome to the game world!'); // Create a popup with the title "Welcome"
    CustomPopup.draw('Tip', 'You have completed the task!', 'custom-style'); // Draw a popup with a custom style class name
    CustomPopup.draw('Attention', 'Non-closable message', '', true); // Create a popup without a close button

2. Set Popup Size

setSize(width, height)

Description: Sets the width and height of the popup.

  • Parameters:
    • width (string): The width of the popup, e.g. '400px'.
    • height (string): The height of the popup, e.g. '300px'.
  • Example:
CustomPopup.setSize('400px', '300px'); // Set the size of the popup to 400x300 pixels

3. Show and Hide Popup

showPopup()

Description: Shows the popup.

  • Example:
CustomPopup.show(); // Show the popup

hidePopup()

Description: Hides the popup.

  • Example:
CustomPopup.hide(); // Hide the popup

4. Set Popup Content

setContent(content)

Description: Updates the content of the popup.

  • Parameters:
    • content (string): The content to update, can be an HTML string.
  • Example:
CustomPopup.setContent('
Task updated: Find the hidden treasure!
'); // Update the popup content

5. Destroy Popup

destroyPopup()

Description: Destroys the popup, removing the popup and the background overlay.

  • Example:
CustomPopup.close(); // Destroy the popup

6. Create and Show Popup

createPopup(options)

Description: Creates and shows a popup based on the provided options.

  • Parameters:
    • options (object): An object containing the following optional properties:
      • title (string): The title of the popup.
      • content (string): The content of the popup.
      • noBtn (boolean): Whether to hide the close button.
      • width (string): The width of the popup.
      • height (string): The height of the popup.
  • Example:

    CustomPopup.create({
        title: 'Welcome',
        content: '
Welcome to the game world! Enjoy your journey.
',
        width: '400px',
        height: '300px'
    }); // Create and show a popup with custom size

Default CSS Styles

In the general.css file, the default styles for CustomPopup are defined as follows:

  • Background Overlay Style (#customPopupBG):

    #customPopupBG {
        position: fixed;
        width: 100%;
        height: 100%;
        background-color: rgb(41 41 41 / 50%);
        z-index: 1000;
        left: 0;
        top: 0;
        backdrop-filter: blur(2px);
    }
  • Popup Container Style (#customPopup):

    #customPopup {
        display: flex;
        flex-direction: column;
        position: fixed;
        top: 30%;
        z-index: 1001;
    }
  • Default Popup Style (.custom-popup):

    .custom-popup {
        background-color: rgba(80, 80, 80, 0.9);
        border: 1px solid rgb(120, 120, 120);
        min-width: 250px;
        min-height: 180px;
        box-shadow: 6px 10px 8px 3px rgb(0 0 0 / 60%);
        backdrop-filter: blur(6px);
    }
  • Title Bar Style (#PopupBanner):

    #PopupBanner {
            width: 100%;
            display: flex;
            height: 24px;
    }
    .popup-title {
            flex: 1;
            text-align: center;
            color: #cbffff;
            background: rgb(89 101 104 / 50%);
    }
    #PopupBanner .closeBtn {
            background-color: rgb(50, 50, 50);
            color: cyan;
            border: none;
            width: 24px;
            height: 24px;
    }

Customizing Popup CSS

If you want to customize the style of the popups, you can follow these steps:

  1. Create a Custom Style Class

    .my-custom-popup {
         background-color: rgba(0, 0, 128, 0.8);
         border: 2px solid #ffcc00;
         box-shadow: 4px 8px 6px 2px rgba(0, 0, 0, 0.5);
         border-radius: 10px;
    }
  1. Apply the Custom Style when Creating the Popup

    CustomPopup.create({
     title: 'Custom Style Popup',
     content: '
This is a popup with a custom style.
',
     width: '350px',
     height: '250px'
    });
    const popup = document.getElementById('customPopup');
    popup.classList.add('my-custom-popup'); // Apply the custom style

Specific Use Cases

  1. Welcome Popup: Display a welcome message at the start of the game.

    CustomPopup.create({
     title: 'Welcome to the Adventure World',
     content: '
Brave adventurer, are you ready to start your journey?
',
     width: '400px',
     height: '200px'
    });
  1. Task Completion Notification: Display reward information when a player completes a task.

    CustomPopup.create({
     title: 'Task Completed!',
     content: '
Congratulations on completing the task! You have earned 500 coins!
',
     width: '300px',
     height: '180px'
    });
  1. Warning Message: In certain important situations, inform the player that a certain action is not allowed.

    CustomPopup.create({
     title: 'Warning',
     content: '
You cannot enter this area, danger is approaching!
',
     noBtn: true,
     width: '350px',
     height: '150px'
    });

With the above features and style customization options, you can create a unique UI experience for your Mod, making the information in the game more rich and eye-catching.
